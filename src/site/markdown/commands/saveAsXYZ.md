### SaveAsXYZ

Save the system as an XYZ file.

---
```
Usage: ffxc SaveAsXYZ [-h] [-n=0] [-p=0] [-r=<scalar>] files...
Save the system as an XYZ file.
    files...            The atomic coordinate file in PDB or XYZ format.
-h, --help              Print this help message.
-n, --neg-offset=0      Negative offset of atom types in the new file.
-p, --pos-offset=0      Positive offset of atom types in the new file
-r, --random=<scalar>   Apply a random Cartesian SymOp with translation range -X .. X.
```
